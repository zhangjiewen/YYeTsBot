# requirements

* tornado
* mongodb
* pymongo

# 导入数据

## 方法1: 自己迁移数据

1. 下载[MySQL的数据](https://t.me/mikuri520/668)
1. 导入数据到MySQL
2. 运行 `python prepare/convert_db.py`
